Verify Internet Connectivity

Under normal circumstances, you should not see this page. There are three main reasons why you may see it:

  • The application is unable to access the Internet
  • The application is unable to obtain results from the NHTSA web service
  • You clicked the Previous button from the "Step 3: Data Download" page

There are two different error messages that you might see:

  • Please verify Internet connectivity and click Try Again
  • Please verify configuration settings and click Try Again

In the first error, "verify Internet connectivity", the application is unable to access the Internet in general. If you are unaware of what may be causing this error, your organization's IT personnel may be able to assist with this problem. Once your computer has Internet access, click the Try Again button. You can also try to run this application from another computer if the error persists.

With the second error, "verify configuration settings", the application is able to access the Internet but not specifically the NHTSA online database. Generally, you or your organization's IT personnel can submit an Issue on the github page for this project and this program's author(s) will be notified to fix the problem. Advanced users can access the configuration settings by clicking on the Configuration expander to attempt a workaround. Explaining the configuration settings is beyond the scope of this help document.
