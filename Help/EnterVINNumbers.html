Step 1: Enter VIN Numbers

In this step, you choose one of two methods for entering VIN numbers:

  • Read VIN numbers from an Excel file
  • Copy and paste them into a text box

Excel:

To read VIN numbers from an Excel file, simply click the Browse button and select the Excel file containing the VIN numbers that you need to look up. Once open, a drop-down box with a list of worksheets in the file will be enabled. Once a worksheet is selected, a drop-down with a list of columns in that worksheet will be enabled. Once a column is selected, you will be able to proceed to the next step.

Note that this application will attempt to automatically select a worksheet and column if it can find a column containing VIN numbers in one of the worksheets. If this happens, please verify the selection before proceeding.

Copy/Paste:

To use the copy and paste method, first copy the VIN numbers using any application. Then, click on the text box labelled "Paste the VIN Numbers into the Following Text Box" and either press CTRL+V or right-click and select Paste.

Note that the VIN numbers being pasted must either be one-per-line or they must be separated by commas or semicolons.
